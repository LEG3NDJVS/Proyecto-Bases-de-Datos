# 🎮 Chocolate-Doom Telemetry & UX Database

Proyecto académico de Bases de Datos enfocado en el diseño e implementación de una base de datos relacional para el análisis de telemetría en partidas de DOOM 🕹️ y métricas de experiencia de usuario (UX) 📊.

El sistema permite almacenar y procesar información como:

* 📍 Trayectorias y movimiento de jugadores.
* ⚔️ Eventos de combate y estadísticas en tiempo real.
* 👥 Análisis de proximidad y cooperación entre jugadores.
* 🧠 Resultados de encuestas UX (PENS, GUESS o BANGS).
* 📂 Procesos ETL para carga y validación de logs TSV.
* ⚡ Consultas analíticas avanzadas, índices y vistas en PostgreSQL.

El objetivo principal del proyecto es identificar patrones de comportamiento, cooperación y posibles sabotajes entre jugadores mediante técnicas de modelado de datos, optimización SQL y análisis de grandes volúmenes de telemetría.
